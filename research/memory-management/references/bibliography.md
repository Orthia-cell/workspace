# Bibliography: Memory Management Research

## Primary Sources

### Foundational Works

1. **Ebbinghaus, H.** (1885). *Über das Gedächtnis* (On Memory). 
   - First empirical study of memory retention and forgetting
   - Established the forgetting curve and savings method
   - Used nonsense syllables to isolate pure memory formation

2. **Wozniak, P.** SuperMemo Algorithm and Spaced Repetition Research.
   - Developed computational models of optimal review timing
   - Created the first digital spaced repetition system
   - Influenced modern learning apps (Anki, Duolingo)

3. **Roediger, H.L. & Karpicke, J.D.** (2006). Test-Enhanced Learning.
   - Demonstrated retrieval practice superiority over passive review
   - Key finding: 80% retention with testing vs. 34% with rereading

### Recent Research (2020-2026)

4. **University of Cambridge, Department of Computer Science** (2020).
   *Adaptive Forgetting Curves for Spaced Repetition Language Learning.*
   - Neural network models for personalized retention prediction
   - Word complexity as highly informative feature
   - URL: ar5iv.labs.arxiv.org/html/2004.11327

5. **Penn State University** (2025). 
   *Fostering Creativity in Science Education Reshapes Semantic Memory.*
   - Network science methods applied to semantic memory
   - "Small-world" network topology in knowledge organization
   - Increased interconnectedness enables creative thinking

6. **IFIByNE-CONICET & University of Buenos Aires** (2025).
   *Remembering One Thing Can Make You Recall More Than You Realize.*
   - Communications Psychology publication
   - Contextual binding and associative memory strengthening
   - Well-established memories reinforced through association

7. **Springer** (2020).
   *Prioritization in Visual Working Memory Enhances Memory Retention.*
   - Retro-cuing paradigm research
   - Dynamic prioritization after encoding
   - Mechanisms: strengthening cued items vs. removing uncued items
