What is SQLite?

SQLite is a serverless, self-contained SQL database engine that reads and writes directly to a single disk file. Unlike MySQL or PostgreSQL, it requires no separate server process — making it ideal for embedded applications, mobile devices, and now potentially... the Orthia workspace.
